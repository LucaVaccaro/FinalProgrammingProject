# FinalProgrammingProject

This final project with 7 classes, Address, Department, Student, Assignment, Course, Gender, Util. Address class stores address details street number, street, city, province, postal code, and country and validates postal codes. Department class represents a department with an ID and name. It also validates department names to ensure they contain only letters and spaces. Assignment class represents an assignment with details such as ID, name, weight, max score, and scores and generates random scores for students based on predefined ranges. Course class manages course details, including ID, name, credits, associated department, assignments, registered students, and final scores. It validates assignment weights and handles student registration and calculates and displays student scores. Student class represents a student with an ID, name, gender, address, and department and manages course registration and dropping, ensuring assignments and scores are updated.
